# GitHub Pages Setup

Das Repo `Tenensounds/edge-patch-advisor` ist fuer GitHub Pages per GitHub Actions vorbereitet.

## Naechste Schritte auf GitHub

1. Das Repo auf GitHub oeffnen.
2. `Settings` -> `Pages` aufrufen.
3. Als Source `GitHub Actions` verwenden, falls GitHub das nicht schon automatisch gesetzt hat.
4. Im Tab `Actions` den Workflow `Deploy GitHub Pages` pruefen.

## Erwartete URL

Nach erfolgreichem Deploy ist die App voraussichtlich hier erreichbar:

`https://tenensounds.github.io/edge-patch-advisor/`
