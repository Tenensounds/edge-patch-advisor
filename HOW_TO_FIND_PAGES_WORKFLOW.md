# How To Find The Pages Workflow

Wenn du den Workflow `Deploy GitHub Pages` nicht sofort siehst:

1. Oeffne im Repo den Tab `Actions`.
2. Suche links in der Workflow-Liste nach `Deploy GitHub Pages`.
3. Falls dort noch nichts erscheint, oeffne `Settings` -> `Pages` und stelle sicher, dass als Source `GitHub Actions` gesetzt ist.
4. Danach erneut `Actions` oeffnen.
