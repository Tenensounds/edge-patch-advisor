# Behringer EDGE Patch Advisor

Eine minimalistische Web-App fuer inspirierende Patch- und Parameter-Vorschlaege fuer den Behringer EDGE.

## Enthalten

- Sound-Kategorien wie Kick Drum, Bassline, Acid-artige Sequenz und Experimental Groove
- Zufaellige Patch-Vorschlaege pro Kategorie
- Parameter-Startwerte fuer wichtige EDGE-Regler
- 8-Step-Sequencer fuer Pitch und Velocity
- Grafische Patch-Kabel zwischen Patchbay-Punkten
- Performance-Tipps fuer Live-Tweaks

## Lokal nutzen

Einfach `index.html` zusammen mit `style.css` und `app.js` lokal bereitstellen und im Browser oeffnen.

## Hinweis

Alle Werte sind Startpunkte und sollten nach Gehoer an dein Setup, Tempo und den Mix angepasst werden.
